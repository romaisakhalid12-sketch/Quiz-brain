# Quiz-brain
Challenge your knowledge with this interactive **True or False Quiz Game** built in Python! Answer a series of interesting questions, receive instant feedback, and see your final score at the end.

## 🎯 About the Project

This quiz game presents players with a series of **True/False** questions. After each answer, the game checks whether it is correct, updates the score, and displays the current progress. Once all questions have been answered, the final score is shown.

## ✨ Features

- ❓ Multiple True/False questions
- ✅ Instant feedback after each answer
- 📊 Real-time score tracking
- 🏆 Final score displayed at the end
- 🧩 Uses Object-Oriented Programming (OOP)
- 🐍 Built with Python
